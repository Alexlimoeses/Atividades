Sistema de Gestão de Estoque - Ferramentas Elétricas

Este projeto é um sistema completo para gestão de estoque de uma loja de ferramentas elétricas, desenvolvido em um único arquivo Python.

Sobre o Projeto
O sistema vai além do básico de adicionar e remover itens. Ele implementa funcionalidades eficientes de busca e ordenação utilizando estruturas de dados clássicas e introduz um recurso inovador: o agendamento automático de manutenções preventivas no momento da venda. Isso permite à loja oferecer um serviço de pós-venda, gerando novas oportunidades de negócio e fidelizando clientes.

Toda a lógica, incluindo as classes, a interface de linha de comando e os testes de unidade, está contida em um único arquivo para facilitar a portabilidade e a execução.

Funcionalidades Principais

Adicionar e Remover Produtos: Gerenciamento básico do inventário.
Busca Rápida: Acesso instantâneo a produtos pelo código.
Listagem Ordenada: Visualização do estoque ordenado por código, nome ou quantidade.
Registro de Vendas: Baixa no estoque e início do ciclo de manutenção.
Fila de Manutenção: Controle das próximas ferramentas a serem revisadas.
Testes Integrados: Verificação da integridade do sistema.
